**Currently building** — the [Decentralized Universal Compute Protocol (DUCP)](https://github.com/ducp-protocol):
[white paper v0.2.0](https://github.com/ducp-protocol/spec/blob/main/whitepaper/DUCP_White_Paper_v0.2.0.pdf),
[specification](https://github.com/ducp-protocol/spec/tree/main/spec), and a
[Rust reference node](https://github.com/ducp-protocol/ducp-node-rs). Companion standard:
[The Quant (ℚ) v0.1.0](https://github.com/ducp-protocol/spec/blob/main/quant/Quant_Standard_v0.1.0.pdf)
— computational energy efficiency, benchmarked against the Landauer limit (open RFC).

**Focused on** — decentralized and verifiable compute, the economics of compute and energy,
mechanism design, and the infrastructure behind large-scale intelligence.

**Open to collaboration** — both DUCP and Quant are Requests for Comments. Critique and
validated benchmark data especially welcome:
[spec issues](https://github.com/ducp-protocol/spec/issues) ·
[`quant`-labelled issues](https://github.com/ducp-protocol/spec/issues?q=label%3Aquant)
